# React & manate Counter Example

A demo Counter project powered by [manate](https://github.com/tylerlong/manate).

### [Live demo](https://chuntaoliu.com/manate-demo-counter/)

## Setup

```
yarn install
```

## Serve

```
yarn serve
```

## Release

```
yarn release
```
